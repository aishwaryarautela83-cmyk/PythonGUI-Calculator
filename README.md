# PythonGUI-Calculator
My First Phython GUI project showcasing a calculator built with Tkinter.This project helped me learn GUI design , event handling and python programming fundamentals.

## 🚀 Features
- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🧹 Clear button
- 🖥️ Simple and user-friendly interface

## 🛠️ Technologies Used
- 🐍 Python
- 🎨 Tkinter

## ▶️ How to Run
1. Install Python.
2. Download or clone this repository.
3. Run:
   
   bash
   python CALCULATOR.py
